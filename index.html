Règles du jeu :

Déplacement :
  • Seul les cases vertes sont accessibles, cliquez dessus pour vous déplacer.
  • Lorsqu'un joueur se déplace sur une arme il récupère les points d'attaque correspondant.
  • Si vous avez déjà une arme vous échangez avec celle présente sur la case.
Combat :
  • Un combat est lancé lorsque les deux joueurs sont côte à côte.
  • Attaque : vous infligez la totalité de vos points d'attaque à votre adversaire.
  • Défense : vous divisez par 2 l'attaque suivante de votre adversaire.
